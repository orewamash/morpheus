What Morpheus Does

🔍

Execution Tracing

Captures every function call, return value, and variable with sys.settrace — no code modification needed.

📖

AI Narration

Uses a local Ollama LLM (or cloud OpenRouter) to narrate execution flow in natural English.

⚠️

Prophecy Mode

Detects rapid repeated calls, missing return values, and other execution warnings before they become bugs.

🔒

Spy Mode

Security-scan unknown scripts for dangerous behavior — file access, network calls, system commands.

🧠

Oracle Mode

Multi-language support (Python, JS, C++, Java) with personality-driven analysis: mentor, critic, paranoid, teacher.

📊

ML Pipeline

Anomaly detection, crash prediction, execution fingerprints, performance degradation tracking, and GAT models.

Get Started in 30 Seconds

1

Install Morpheus

Requires Python 3.11+ and Ollama (free, local LLM).

pip install morpheus-cli
2

Install & start Ollama

Download from ollama.com, then pull a model.

ollama pull mistral
3

Trace any Python file

Morpheus runs your script, captures every event, and narrates it.

morpheus run examples/simple.py
4

Explore more modes

Security scan, crash prediction, teaching mode, and interactive mind maps.

morpheus spy script.py  // morpheus run script.py -m prophecy
