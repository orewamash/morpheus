What Morpheus Does

Six powerful modes to understand, analyze, and improve your code.

🔍

Execution Tracing

Captures every function call, return value, and local variable using Python's sys.settrace — zero code modification required.

Core Engine
📖

AI Narration

Uses a local Ollama LLM (or cloud OpenRouter) to narrate your execution flow in natural, readable English — like a senior dev walking through the code.

Default Mode
⚠️

Prophecy Mode

Proactively detects rapid repeated calls, missing return values, and anomalous patterns — catching bugs before they happen.

Predictive
🔒

Spy Mode

Security-scan unknown scripts for dangerous behavior — file access, network calls, and system commands — before you run them.

Security
🧠

Oracle Mode

Multi-language support (Python, JS, C++, Java) with four personalities: mentor, critic, paranoid, teacher.

Multi-Lang
📊

ML Pipeline

Anomaly detection (Isolation Forest), crash prediction (Random Forest), execution fingerprints, performance degradation, and GAT models.

Advanced

Get Started in 30 Seconds

Four steps from zero to your first execution narration.

01

Install Morpheus

Requires Python 3.11+. Optionally install ML extras.

pip install morpheus-cli
02

Start Ollama

Download from ollama.com, then pull a model.

ollama pull mistral
03

Run your first trace

Morpheus runs the script and narrates everything.

morpheus run examples/simple.py
04

Explore the modes

Security scan, teach mode, oracle, mind maps, and ML analysis.

morpheus spy script.py

Every Mode Explained

Morpheus has a mode for every workflow.

run

Narrator

LLM narrates each execution chapter in plain English

run -m prophecy

Prophecy

Detects rapid calls, missing returns, and execution warnings

run -m teach

Teach

Interactive Q&A session about the code execution

run -m oracle

Oracle

Multi-language + personality-driven analysis

spy

Spy

Security scan unknown scripts for dangerous behavior

map

Mind Map

Interactive execution graph visualization dashboard

analyze

ML Analyze

Full ML pipeline on saved runs — anomalies, degradation, profiles

diff

Time Diff

Compare two runs and see where execution diverged
